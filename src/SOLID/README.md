Principles Vs Design Patterns
## Principle:
    General guidelines or the rules of thumb for writting good code

- SOLID
- DRY
- KISS
- YAGNI

They are you should strive for, but not how to implement them.

## Design patters
    Reusable solutions to common design problem

- Principles are abstract, not concrete
- You can not achieve next, until previous
